## 画像ファイル情報

このフォルダには以下の画像ファイルを配置してください。
ウェブサイトの完全な表示のためには、以下のファイルが必要です：

1. creator/character.png - メインキャラクター画像（推奨サイズ: 500x500px）
2. creator/sample-creator.png - サンプルクリエイター画像（推奨サイズ: 400x400px）
3. company/logo.png - FunLinkロゴ（推奨サイズ: 200x60px）
4. company/sample-company.png - サンプル企業ロゴ（推奨サイズ: 200x200px）
5. icons/favicon.ico - ファビコン（推奨サイズ: 32x32px）

画像ファイルがない場合でもウェブサイトは表示されますが、一部の要素が正しく表示されません。
画像は独自に作成するか、無料の画像リソースを利用してください。

※著作権に注意: 商用利用する場合は、適切なライセンスを持つ画像を使用してください。
